Upgrade Instructions - for PHP-Nuke Platinum 7.6.b.4
Step by Step Guide

 
How to Upgrade
Note: The following steps must be done in the sequential order for the upgrade to work.

Step 1
  • BACKUP ALL YOUR SITE FILES AND MAKE A COMPLETE BACKUP OF YOUR DATABASE STRUCTURE AND DATA!!   THIS IS VITAL AND I CANNOT STRESS ENOUGH THE IMPORTANCE OF THIS!!!.

  • Replace ALL your files and folders with the ones included in the public_html folder from the 7.6.b.4 package.
     


Step 2

  • Open the Upgrades folder from the 7.6.b.4 package and find the version of Nuke your site is currently running.  For instance if you are running PHPNuke Version 6.0 then you would open the ~Nuke60-76b2 folder and go into the Nuke60-76 folder and run each update script in sequence all the way up to the 7.6.b.4 script located in the 76b3-76b4 folder.   Just make sure you start at the correct version that your site is running. 
     
  • Since there were more than one version of 7.6.b.3 you might run into a problem with the b3TOb4 upgrade script.   If you have already added the protector patch to your b3 site then you will need to first run the afterProtectorpatch.sql script located in the 76b3-76b4/SQL folder through phpmyadmin PRIOR to running the b3TOb4 script.  If you get an error message that the forum_notify column does not exist when running this script then open the b3TOb4.sql file and remove this line at the top ALTER TABLE nuke_bbforums DROP forum_notify; Resave it and run it through again.

  • Once you have ran all the upgrade scripts, then reenter the database info into your config.php file.

Step 3
  • Go to your site now.   Due to the amount of changes, any Mods you have added to the original package will most likely have to be readded again.   You should have retained all your users, forums, and posts.    You will also need to reenter your config info in your Admin - Preferences and your Forums Admin - General Admin -Configuration Areas.  

Good Luck
