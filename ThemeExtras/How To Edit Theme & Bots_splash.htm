How to upload SDDesign Themes?
   

Unzip the zip file containing the theme & or goodies using WinZip.

Upload theme folder maintaining their nuke "root" directory structure. For example we are going to use BF2 Theme, need to be uploaded to /public_html/theme folder.

If you are "not" using your "root directory" it would need to be in your PHP Nuke /theme directory structure.

For General Knowledge on how to set up your SDDesign Themes http://www.sddesign.biz/forums-kb-article-12.html

 

How to use SDDesign Themes Editing?
   

Upload .txt file in the main "root" directory structure. where your config.php and mainfile.php. For example we are going to use BF2 Theme, then you should  be uploading the bf2.txt to your nuke "root" directory structure. To edit your .txt file right click and open with "notepad" then..

 
   Edit the link names and their corresponding URL's are very simple.

 For example:

&button1=Home&
&url1=/html/index.php&

or

&button1=Home&
&url1=/index.php&


The above codes Point to the link Home and also adds the corresponding URL's to the button.

You can edit this to anything you desire. For example:

&button1=MynewWebsite&
&url1=/abc/mynewwebsite.php&

You can edit the Site name, Slogan, Button names, Button URL's in the header & footer with this.

  How to use SDDesign Bots_Splash?
   

Unzip the Bots_splash_540.zip

info.txt <------ File to Edit.

How To Edit Theme & Bots_splash.htm

Edit the link names and their corresponding URL's are very simple. For example:

&button4=Enter&
&url4=/index.php&

&button3=Login to check your Pvt msg.&
&url3=/modules.php?name=Your_accounts&

The above codes Point to the link Home and also adds the corresponding URL's to the button.
  Open sitenews.txt and you can edit this to anything you desire. For example:

&sitenews=Welcome to sddesign.biz.This is a promotional shoutbox given to the sddesign users for free. Edit the shoutbox to your hearts content. You can put your sitename and slogan in the splash page simply by editing the notepad.&

You can edit the above &sitenews= "edit what you desire here" &

 
  Now let's Edit your Bots_splash.html just right click and choose open with "notepad". then FIND, REPLACE, ADD below.

FIND:
<meta name="description" content="Bot_splash">

REPLACE:
Bot_splash

ADD:
index


FIND:
id="Bot_splash" width="100%" height="100%">

REPLACE:
Bot_splash

ADD:
index


FIND:
<param name="movie" value="Bot_splash.swf">

REPLACE:
Bot_splash.swf

ADD:
index.swf



FIND:
name="Bot_splash" src="Bot_splash.swf"

REPLACE:
Bot_splash <--both--> Bot_splash.swf

ADD:
index.swf

Save and Close file and upload it to your root directory of your PHP Nuke installation directory.

 
  NOTE: If u want to run it as index, you need to rename those files to index.html and index.swf and then put it in the root directory of your PHP Nuke installation.

But if u want to use it as something else, then u can name it whatever u want to just make sure that when you rename it that you have to edit those file with what you have renamed Bot_splash.html file.
